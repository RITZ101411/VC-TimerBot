# VC-TimerBot

寝落ちを予約して時間経過で通話から切断するだけのDiscord BOT

# commands

| コマンド名     | 説明                                 | 使用例                         |
|----------------|--------------------------------------|--------------------------------|
| `/sleep-timer`       | タイマーをセットする時間（分） | `/sleep-timer 10` |
| `/check-timer`        | 指定したユーザーのタイマーを確認します | `/check-timer RITZ`                        |
| `/cancel-timer`       | 現在のタイマーをキャンセルします     | `/cancel-timer`                       |
